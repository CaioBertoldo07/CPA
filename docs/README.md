# Documentacao do Sistema CPA

Esta pasta centraliza a documentacao tecnica do projeto.

## Conteudo

- [Diagrama ER e modelo de dados](./database.md)

## Escopo

A documentacao de banco descreve:

- Entidades principais do sistema de avaliacao
- Relacionamentos entre questoes, avaliacoes, cursos e respostas
- Tabelas auxiliares de classificacao (eixos, dimensoes, categorias, modalidades)
- Observacoes sobre tabelas ignoradas no Prisma Client
